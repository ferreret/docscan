# :material-keyboard: Atajos de teclado

## Workbench

| Atajo | Acción |
|-------|--------|
| `Ctrl+S` | Escanear |
| `Ctrl+I` | Importar ficheros |
| `Ctrl+T` | Transferir lote |
| `Ctrl+P` | Reprocesar página |
| `←` / `→` | Página anterior / siguiente |
| `Home` / `End` | Primera / última página |
| `Ctrl+→` | Siguiente con barcode |
| `Ctrl+Shift+→` | Siguiente pendiente de revisión |
| `Ctrl+W` | Cerrar lote |

## Visor

| Atajo | Acción |
|-------|--------|
| `Ctrl++` / `Ctrl+-` | Zoom in / out |
| `Ctrl+F` | Ajustar a ventana |
| `Ctrl+0` | Zoom 100% |
| `R` | Rotar 90° derecha |
| `Shift+R` | Rotar 90° izquierda |
| `M` | Marcar/desmarcar página |
| `Del` | Eliminar página |
