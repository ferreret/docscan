# Progreso 2026-03-24

## Resumen

Implementado el **Asistente IA "GOD MODE"** — un asistente conversacional unificado a nivel de Launcher que permite crear, modificar, duplicar y eliminar aplicaciones completas de DocScan Studio mediante lenguaje natural.

## Cambios realizados

### Merge release_2b → main
- Merge de la rama `release_2b` (verification panel) a `main`
- Push a remoto
- Creada nueva rama `release_3` para el asistente IA

### Asistente IA — Servicio core
- **`app/services/_assistant_constants.py`**: Constantes compartidas (IMAGE_OPS, SCRIPT_API, EVENT_SIGNATURES)
- **`app/services/pipeline_assistant.py`**: Servicio per-pipeline con tool calling (Anthropic/OpenAI) — creado inicialmente, luego refactorizado para extraer constantes
- **`app/services/god_mode_assistant.py`**: Servicio GOD MODE con 7 tools:
  - `list_applications` — listar apps
  - `get_application` — config completa de una app
  - `create_application` — crear app desde cero
  - `update_application` — modificar campos parciales
  - `duplicate_application` — clonar con modificaciones
  - `delete_application` — eliminar app
  - `set_event_code` — generar codigo de eventos lifecycle

### Asistente IA — UI
- **`app/ui/launcher/god_mode_panel.py`**: Panel chat lateral en el Launcher con:
  - Selector de proveedor (Anthropic/OpenAI)
  - Input de API key inline
  - Area de chat con burbujas
  - Preview de cambios con Aceptar/Rechazar por cada operacion
  - Ejecucion BD (create/update/duplicate/delete) al aceptar
  - Historial de conversacion con resumen de tool calls
- **`app/ui/launcher/app_change_preview.py`**: Widget preview estructurado por tipo de operacion
- **`app/ui/launcher/launcher_window.py`**: QSplitter + boton "GOD MODE" en toolbar

### Asistente IA — Workers y widgets auxiliares
- **`app/workers/god_mode_worker.py`**: QThread para llamadas API
- **`app/workers/pipeline_assistant_worker.py`**: QThread para pipeline (mantenido)
- **`app/ui/configurator/pipeline_diff_widget.py`**: Widget diff visual (mantenido, reutilizable)
- **`app/ui/configurator/pipeline_assistant_panel.py`**: Panel per-pipeline (mantenido)
- **`app/ui/configurator/event_assistant_dialog.py`**: Dialogo per-evento (mantenido)

### Limpieza per-tab
- `tab_pipeline.py`: Eliminado panel IA lateral (ahora GOD MODE en launcher)
- `tab_events.py`: Eliminado boton IA (ahora GOD MODE en launcher)

### Estilos
- dark.qss, light.qss, default.qss: Estilos para GOD MODE panel, chat bubbles, preview widgets

### Tests
- **`tests/test_pipeline_assistant.py`**: 30 tests del servicio pipeline
- **`tests/test_god_mode_assistant.py`**: 21 tests del servicio GOD MODE
- **Total tests**: 769 passing, 0 failed

## Bugs corregidos durante la sesion
- **Historial de conversacion roto**: El AI repetia acciones porque los tool calls no se registraban en el historial. Arreglado con `_summarize_tool_call()` y registro de aceptacion/rechazo.
- **auto_transfer activado por defecto**: Corregido en el system prompt para que sea false por defecto.
- **Scripts innecesarios**: Mejorado system prompt con reglas de minimizacion de ScriptSteps y documentacion completa de transfer config, filename_pattern, regex de BarcodeStep.
