### Detalles técnicos

**FolderWatcher** (`folder_watcher.py`):
- Usa `FileClosedEvent` (IN_CLOSE_WRITE en Linux) para garantizar ficheros completamente escritos
- `EventDebouncer` agrupa ráfagas de ficheros en un solo lote
- Modo centinela: espera fichero específico (ej: GO.txt) antes de procesar
- APScheduler `BackgroundScheduler` para limpieza periódica y reintento de errores
- `PatternMatchingEventHandler` filtra por extensión (jpg, png, tif, pdf, bmp)

**Worker** (`worker_main.py`):
- Dos modos: `--watch` (folder-watch continuo) y `--process-pending` (procesa lotes en BD y sale)
- Reutiliza PipelineExecutor, BatchService, TransferService del core
- Compila scripts de pipeline y eventos de ciclo de vida al inicio
- Shutdown coordinado con SIGINT/SIGTERM via threading.Event
- Auto-transfer si configurado en la aplicación
- Estadísticas de procesamiento guardadas en batch.stats_json

### Mejora de calidad de renderizado B/N

- **Problema**: Las imágenes binarias (1-bit, valores 0/255) se veían pixeladas en el visor porque la interpolación bilineal de Qt no genera grises intermedios sobre fuente binaria.
- **Solución**: `_antialias_binary()` en `page_state.py` aplica Gaussian blur 3x3 (sigma=0.8) a imágenes binarias sin cambiar dimensiones. Crea valores grises intermedios en bordes de texto que Qt interpola correctamente con `SmoothTransformation`.
- `QGraphicsPixmapItem.setTransformationMode(SmoothTransformation)` aplicado directamente al item.
- `LosslessImageRendering` render hint añadido.
- No se cambian dimensiones de imagen → overlays de barcodes y fit-to-page funcionan correctamente.

## Uso

```bash
# Worker folder-watch (modo continuo)
python3.14 -m docscan_worker --app-name "MiApp" --watch /ruta/entrada

# Worker folder-watch con centinela
python3.14 -m docscan_worker --app-name "MiApp" --watch /ruta/entrada --sentinel GO.txt

# Procesar lotes pendientes y salir
python3.14 -m docscan_worker --app-name "MiApp" --process-pending

# Lanzar app directamente (con UI)
python3.14 main.py "MiApp"

# Modo directo headless (escanea + transfiere)
python3.14 main.py --direct-mode "MiApp"
```
