# Progreso 2026-03-25

## Resumen

Sesión muy productiva con múltiples features implementadas, refactorizaciones de UI y limpieza de código. Se completaron todas las features funcionales pendientes del roadmap.

## Cambios realizados

### Renombrado God Mode → AI Mode
- Renombrados todos los ficheros, clases, variables y estilos QSS
- `god_mode_assistant.py` → `ai_mode_assistant.py`, `GodModePanel` → `AiModePanel`, etc.
- Eliminados ficheros antiguos

### Mejoras al AI Mode
- Documentado `ai_config` (barcode_regex, barcode_fixed_value) en el system prompt
- Añadidos ejemplos de `ImageLib.merge_to_tiff()` y `merge_to_pdf(append=True)`
- Instrucciones explícitas para merge en vez de contador numérico
- Auditoría completa de campos: añadido schema tipado para `transfer` (17 propiedades)
- Documentados `index_fields`, `default_tab`, `output_bw_threshold`, `collision_policy`

### Optimizaciones AI Mode (/simplify)
- `_build_apps_summary()`: reemplazado `deserialize()` por `json.loads()` en UI thread
- System prompt: parte estática pre-renderizada a nivel de módulo
- OpenAI tools: pre-computadas como `_OPENAI_TOOLS`
- Worker: `configure()` solo cuando cambia provider/key
- Worker: emite error si está ocupado en vez de ignorar silenciosamente

### Test Pipeline (IMG-14)
- `app/pipeline/test_executor.py`: `InstrumentedPipelineExecutor` + `StepSnapshot`
- `app/workers/test_pipeline_worker.py`: QThread para test
- `app/ui/configurator/test_pipeline_dialog.py`: Diálogo de resultados por paso
- Botón "Probar pipeline" en `tab_pipeline.py`
- 9 tests

### Export/Import aplicación como JSON (CFG-02/03)

- Formato `.docscan` (JSON legible, version 1)

- 18 tests

### Sidebar colapsable para el Launcher

### Splash Screen
- `app/ui/splash_screen.py`: QPainter con degradado oscuro, logo SVG, versión, progreso
- Integrado en `main.py` con mensajes de estado durante la carga

## Decisiones de diseño
- **ConditionStep y HttpRequestStep descartados**: redundantes con ScriptStep + AI MODE
- **Perfiles de usuario (LCH-07/BAT-05) descartados**: complejidad innecesaria
- **batch.lock (BAT-03) descartado**: foco mono-estación
- **Deshabilitar botones navegación (APP-08) descartado**: mejora cosmética menor
- **Pipeline templates (IMG-13) aplazado**: AI MODE + Export/Import cubren el caso de uso
